Uygulamayı çalıştırarak denemek istiyorsanız 
1-) app:build.gradle doosyasında bulunan siginconfig işlemlerini yeni bir keystore oluşturmalısınız.
2-) Huawei Console(App Gallery Connect) üzerinden proje oluşturup, analytics ve ml kitlerini aktifleştirerek agconnect-services.json dosyasını güncelleyerek build alabilirsiniz.
3-) Firebase tarafındada uygulama bağımlılıklarını karşılayacak şekilde google-services.json dosyasını projeye eklemelisiniz.
